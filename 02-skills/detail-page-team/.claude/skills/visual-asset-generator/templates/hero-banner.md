# 히어로 배너 프롬프트 템플릿

상세페이지 최상단에 배치되는 메인 비주얼 이미지 생성용 프롬프트

## 기본 구조

```
[제품/주제] + [촬영 스타일] + [배경] + [조명] + [분위기] + [기술 요구사항]
```

## 카테고리별 템플릿

### 식품

```
{product_name}, appetizing food photography, traditional Korean earthenware pot, 
steam rising, wooden dining table, warm natural window lighting, 
cozy home kitchen background, overhead angle shot, 
professional food photography, high resolution
```

**키워드**:
- 촬영: appetizing, delicious, mouth-watering
- 배경: wooden table, kitchen, dining room, marble surface
- 조명: natural lighting, warm light, soft shadows
- 분위기: cozy, homely, inviting, fresh

### 뷰티

```
{product_name}, luxury cosmetics photography, elegant marble surface, 
rose petals scattered, soft pink gradient background, 
studio lighting with soft reflections, 
premium beauty product shot, high-end magazine style
```

**키워드**:
- 촬영: elegant, luxurious, premium, sophisticated
- 배경: marble, pink fabric, flower petals, water droplets
- 조명: soft studio light, gradient, subtle glow
- 분위기: feminine, delicate, pure, clean

### 가전

```
{product_name}, modern tech product photography, 
minimalist white studio background, sleek reflective surface, 
cool blue accent lighting, floating shadow effect, 
professional product shot, Apple-style aesthetic
```

**키워드**:
- 촬영: modern, sleek, innovative, cutting-edge
- 배경: white studio, gradient, tech environment
- 조명: studio lights, rim lighting, cool tones
- 분위기: professional, futuristic, clean, premium

### 패션

```
{product_name}, fashion editorial photography, 
neutral beige studio backdrop, dramatic side lighting, 
soft fabric texture visible, elegant composition, 
high fashion magazine style, professional styling
```

**키워드**:
- 촬영: editorial, stylish, trendy, sophisticated
- 배경: studio, lifestyle setting, urban
- 조명: dramatic, fashion lighting, soft box
- 분위기: elegant, modern, aspirational

## 플랫폼별 사이즈 요구

프롬프트 끝에 추가:

```
# 쿠팡/스마트스토어
, aspect ratio 860:500, horizontal banner format

# 마켓컬리
, aspect ratio 1010:600, wide horizontal banner

# 카페24
, aspect ratio 1200:600, full-width banner format
```

## 품질 향상 키워드

항상 끝에 추가 권장:

```
, professional photography, high resolution, 8K quality, 
sharp focus, vibrant colors, commercial quality
```

## 피해야 할 것

- ❌ 텍스트 포함 요청 (AI가 텍스트 생성 못함)
- ❌ 특정 브랜드 로고 요청
- ❌ 유명인/실존 인물 요청
- ❌ 지나치게 복잡한 구성
