# 섹션 배너 프롬프트 템플릿

상세페이지 각 섹션 구분용 배경 이미지 생성

## 특징

- 텍스트 오버레이 공간 확보
- 브랜드 컬러와 조화
- 추상적/비침습적 디자인
- 섹션 주제와 연관된 분위기

## 기본 구조

```
Abstract background for {section_topic}, [스타일] + [컬러] + [요소] + [텍스트 공간]
```

## 섹션별 템플릿

### 혜택 섹션 (Benefits)

```
Abstract background for product benefits section, 
soft gradient from {brand_color} to white, 
subtle geometric shapes, floating circles, 
large empty space in center for text overlay, 
modern clean design, minimal and professional
```

### 특징 섹션 (Features)

```
Abstract technology background for features section,
subtle grid pattern, soft blue gradient,
light geometric lines connecting dots,
clean space for content overlay,
modern tech aesthetic, minimal design
```

### 신뢰 섹션 (Trust/Certification)

```
Abstract background for trust and certification section,
soft golden gradient with white,
subtle shield or badge shape in corner,
elegant texture, premium feel,
large text area in center,
professional and trustworthy mood
```

### 사용법 섹션 (How to Use)

```
Abstract step-by-step background,
soft numbered circular shapes (1, 2, 3) as watermark,
light gradient from left to right,
clean modern design,
space for instructional content
```

### 리뷰 섹션 (Reviews)

```
Abstract background for customer reviews section,
soft warm gradient, conversation bubble shapes as watermark,
friendly and inviting atmosphere,
speech mark patterns very subtle,
clean space for testimonial text
```

### FAQ 섹션

```
Abstract Q&A section background,
subtle question mark shapes as pattern,
soft gradient with brand accent color,
professional and helpful mood,
clean organized feel
```

```
# 마켓컬리

aspect ratio 1200:300, full-width section divider
```

```

```
