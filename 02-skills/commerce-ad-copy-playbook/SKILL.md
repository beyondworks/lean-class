# 커머스 퍼포먼스 광고 카피 플레이북 (매체 무관)

이 플레이북은 **커머스 D2C 전용 영감 카드덱**입니다. 9개 카테고리는 *업종*이 아니라 *소구·이미지·훅 카드의 묶음*으로 보세요. 절대 그대로 복사하지 말고, 매번 다른 변형을 생성하세요. **매체 무관 동일 활용** — 메타·구글·네이버·카카오 모두 같은 소구 패턴 사용 가능. 매체별 글자수·CTA 차이는 [ad-copywriter](.claude/agents/ad-copywriter.md)가 [ad-specs.md](.claude/) 기반으로 자동 분기.

**다루는 카테고리** — 모두 *상품 판매(D2C 커머스)* 영역:
1. 패션 (의류·잡화·슈즈)
2. 뷰티 (스킨케어·메이크업·헤어·바디)
3. 건기식 (건강기능식품·헬스 보조제)
4. 식품 (가공식품·신선식품·간편식·음료)
5. 가전·디지털 (가전제품·IT 기기·가젯)
6. 리빙 (가구·홈소품·키친·인테리어 용품)
7. 다이어트·건강관리 (체중관리 식품·헬스 보조 기기·홈트 도구)
8. 펫 (반려동물 사료·간식·용품)
9. 유아·키즈 (이유식·기저귀·아동복·교육 완구)

## 사용 방법 — 소구 포인트 콜라주

**1:1 카테고리 매칭은 금지**입니다. 실제 D2C 상품은 단일 카테고리에 깔끔하게 들어가지 않습니다 (예: 다이어트 식품은 식품 + 건기식 + 다이어트 카드의 콜라주). 대신 다음 순서로 카드를 조합하세요.

### Step 1. 핵심 소구 식별 (랜딩 분석에서)
랜딩이 약속하는 가치를 **2~4개의 소구 포인트**로 분해하세요. 아래 "소구 축 인덱스"의 표현을 사용하면 매칭이 빠릅니다.

예) 단백질 쉐이크 D2C → [효과/변화] + [편함/간편] + [가성비/혜택] + [신뢰/1위]

### Step 2. 카드 픽업 (각 소구마다 카테고리 1~2개)
각 소구마다 그 소구를 가장 잘 표현하는 카테고리에서 *조각*을 흡수하세요. 한 카테고리 전체를 가져오는 게 아닙니다.

| 흡수할 카드 | 가져오는 것 |
|---|---|
| **소구 어휘**: 카테고리의 "포함 단어"와 어휘 결 |
| **카피 훅 구조**: "카피 예시"의 *문장 구조* (예: "달라집니다" 결과형, "단번에" 즉각형, "오늘부터" 시작형) |
| **이미지 패턴**: 인물 유무·성별·연령대·구도·소품·제품 배치 |
| **CTA**: 1순위 소구의 카테고리에서 우선 채택 |
| **A/B 축**: 1순위 소구의 카테고리 축을 따르되, 다른 소구 카드에서 더 강한 축이 있으면 교체 가능 |

### Step 3. 콜라주 후 재작성
픽업한 카드들을 머릿속에서 합친 뒤, **카피 예시를 그대로 쓰지 말고 단어·리듬·구체 명사를 새로 채워** 헤드라인을 만드세요. 헤드라인은 9~16자 기본, 길어도 22자 이내.

### Step 4. 카드 출처 기록
디자이너에게 brief를 넘기기 전에, "어떤 카테고리에서 어떤 카드를 흡수했는지" 1~2줄로 메모하세요. 디자이너가 톤을 잡을 때 참고합니다. (예: "2번에서 *즉각 변화* 어휘, 3번에서 *1위·신뢰* 강조 패턴, 4번에서 *간편* 어휘 흡수")

## 휴리스틱 — 모든 콜라주 공통

- **헤드라인 첫 단어**가 클릭률을 좌우합니다. 숫자(할인율·1위·후기 수), 의문, 감탄, 명령형, 부정문 중 하나로 시작하는 변형을 항상 1개는 포함하세요.
- **포함 단어**는 픽업한 카드들 어휘에서 1~2개 자연스럽게 녹여 쓰세요. 억지로 다 쓰지 마세요.
- **CTA**는 1순위 소구 카테고리의 명시 CTA를 우선. 커머스는 대부분 **"지금 구매하기" / "구매하기" / "주문하기"** 가 1차 행동. 단, 랜딩의 실제 1차 행동(체험·구독·앱 다운로드 등)과 충돌하면 랜딩 우선 + 변경 이유 한 줄 메모.
- **이미지 지시**는 디자이너가 그대로 외부 이미지 생성/스톡 검색에 쓸 수 있는 수준으로 구체적으로 (성별/연령대/표정/공간/소품/조명/구도). **커머스는 *제품 컷이 핵심* — 실제 랜딩 제품 컷을 합성하는 워크플로우 명시 의무.**
- **혜택·할인·1+1·1위** 등 *수치형 강조*는 커머스 광고의 클릭률 핵심. 랜딩에서 추출 가능한 수치(가격·할인율·후기 수·1위 등)는 모두 [수치 명세]에 모아 SSOT로 활용.
- **카테고리에 없는 결**(예: "신선식품 D2C = 식품 + 건기식 신뢰 + 다이어트 결과 콜라주")이라도 카드 조합으로 만들어낼 수 있어야 합니다. 새 카테고리를 임의로 만들지 마세요. 카드만 새로 조합하세요.

## 카테고리 1. 패션 (의류·잡화·슈즈)
- **이미지**: 실사 / 사람 중심. 착용 컷(전신 or 반신), 자연스러운 일상 or 거리, 스타일 강조(코디 완성), 여성 모델 20~30대 중심(트렌드/감성). 남성 모델은 남성 타깃 시 별도 테스트. 제품 단독 컷(화이트 백)도 보조.
- **카피 예시**: "입는 순간, 분위기가 달라집니다"
- **포함 단어**: 스타일, 핏, 코디, 분위기, 데일리, 신상
- **소구**: 매력 (외형 변화)
- **CTA**: 지금 구매하기 / 구매하기
- **A/B 축**: 이미지 — A: 착용 컷 / B: 제품 단독 / 소구 — A: 스타일 / B: 체형 보정

## 카테고리 2. 뷰티 (스킨케어·메이크업·헤어·바디)
- **이미지**: 실사 / 얼굴 중심(여성). 피부 before → after, 클로즈업(피부결 강조), 사용 장면 or 결과 강조 컷. 밝고 깨끗한 톤. 제품 컷 + 사용자 만족 표정.
- **카피 예시**: "단번에 달라지는 피부"
- **포함 단어**: 효과, 변화, 피부, 개선, 결, 광채
- **소구**: 체감 (즉각적 변화)
- **CTA**: 구매하기 / 지금 구매하기
- **A/B 축**: 이미지 — A: before/after / B: after / 카피 — A: 즉각 효과 / B: 꾸준한 개선

## 카테고리 3. 건기식 (건강기능식품·헬스 보조제)
- **이미지**: 실사 / 사람 + 제품. 건강한 모습(활력 있는 표정), 중장년 or 30~50대 모델, 제품 + 복용 상황. 자연/건강한 배경.
- **카피 예시**: "몸이 먼저 달라집니다"
- **포함 단어**: 건강, 개선, 활력, 케어, 흡수, 면역
- **소구**: 개선 (건강 변화)
- **CTA**: 지금 시작하기 / 구매하기
- **A/B 축**: 소구 — A: 효과 / B: 예방 / 이미지 — A: 사람 중심 / B: 제품 중심

## 카테고리 4. 식품 (가공식품·신선식품·간편식·음료)
- **이미지**: 실사 / 음식 중심. 김 모락 나는 따끈한 컷, 잘 차려진 한 끼, 손이 음식을 집는 동작, 가족 식탁 장면. 신선 식품은 *원물 컷* (당일 수확·재료 클로즈업). 간편식은 *조리 직전·직후* 액션.
- **카피 예시**: "오늘 밥상이 달라집니다" / "이 한 끼면 충분합니다"
- **포함 단어**: 맛, 신선, 한 끼, 든든, 집밥, 간편, 산지직송
- **소구**: 미각·편의 (맛 + 간편)
- **CTA**: 구매하기 / 지금 주문하기
- **A/B 축**: 이미지 — A: 음식 클로즈업 / B: 식탁·라이프스타일 / 소구 — A: 맛 / B: 간편함

## 카테고리 5. 가전·디지털 (가전제품·IT 기기·가젯)
- **이미지**: 실사 / 사람 + 제품. 제품을 *사용 중*인 컷(켜진 화면·작동 인디케이터 명시). 30~40대 모델 자연스러운 일상 공간(거실·주방·서재). 제품 단독 컷은 화이트 백 + 미니멀.
- **카피 예시**: "오늘부터 생활이 달라집니다"
- **포함 단어**: 편리, 변화, 스마트, 자동, 절약, 신기능
- **소구**: 편의·혁신
- **CTA**: 구매하기 / 자세히 보기
- **A/B 축**: 이미지 — A: 사용 장면 / B: 제품 단독 / 소구 — A: 편리 / B: 절약(에너지·시간)

## 카테고리 6. 리빙 (가구·홈소품·키친·인테리어 용품)
- **이미지**: 실사 / 공간 중심. 완성된 인테리어 컷 (제품이 자연스럽게 배치된 거실·침실·주방), 따뜻한 자연광, 손이 제품을 잡는 동작 컷. 여성 20~30대 *라이프스타일 톤*. 비포애프터는 *공간 정리* 컨셉에 한정.
- **카피 예시**: "이 한 점이 공간을 바꿉니다"
- **포함 단어**: 공간, 분위기, 인테리어, 가성비, 일상, 정리
- **소구**: 변화 (공간 분위기 업그레이드)
- **CTA**: 구매하기 / 자세히 보기
- **A/B 축**: 이미지 — A: 공간 컷 / B: 제품 클로즈업 / 소구 — A: 분위기 / B: 실용·정리

## 카테고리 7. 다이어트·건강관리 (체중관리 식품·헬스 보조 기기·홈트 도구)
- **이미지**: 실사 / 여성 중심 (20~40대). before → after 체형 변화 or 운동·식사 후 만족 컷. 활동적 톤(밝은 운동복·자연광). 제품 (단백질 쉐이크·체중계·헬스 기기) 사용 컷.
- **카피 예시**: "눈에 보이는 변화"
- **포함 단어**: 감량, 변화, 체형, 칼로리, 핏, 가볍게
- **소구**: 결과 (체형 변화)
- **CTA**: 시작하기 / 지금 구매하기
- **A/B 축**: 이미지 — A: before/after / B: after 만족 / 소구 — A: 결과 / B: 과정(꾸준함)

## 카테고리 8. 펫 (반려동물 사료·간식·용품)
- **이미지**: 실사 / 반려동물 + 보호자. 교감 장면(쓰다듬기·먹이는 손), 행복한 반려동물 표정, 자연광 거실. 제품(사료·간식) + 반려동물 함께 컷. 20~40대 여성·남성 모두 가능.
- **카피 예시**: "우리집 막둥이를 위한 선택"
- **포함 단어**: 반려, 케어, 영양, 안심, 까다로운, 입맛
- **소구**: 감정·신뢰 (가족 같은 케어)
- **CTA**: 구매하기 / 지금 구매하기
- **A/B 축**: 이미지 — A: 보호자+반려동물 / B: 반려동물 단독 / 소구 — A: 사랑·감정 / B: 영양·신뢰

## 카테고리 9. 유아·키즈 (이유식·기저귀·아동복·교육 완구)
- **이미지**: 실사 / 아이 + 부모(엄마 중심). 놀이·식사·잠 자는 일상 컷, 따뜻한 자연광, 안심 분위기. 제품(이유식·기저귀·완구) 사용 컷. 0~6세 영유아·키즈 모두 가능 (카테고리 세부 타깃에 맞춰).
- **카피 예시**: "아이에게 꼭 맞는 선택"
- **포함 단어**: 안심, 영양, 성장, 부드러움, 까다로운, 엄마
- **소구**: 신뢰·안심 (엄마의 까다로운 기준)
- **CTA**: 구매하기 / 자세히 보기
- **A/B 축**: 이미지 — A: 아이 단독 / B: 부모+아이 / 소구 — A: 안심·성분 / B: 사랑·일상

## 소구 축 인덱스 (역인덱스)

랜딩의 핵심 가치를 아래 소구 축으로 분해한 뒤, 각 축에서 카드를 픽업하세요. 한 축에 여러 카테고리가 묶여 있으면 가장 결이 가까운 카테고리를 1~2개 고르면 됩니다. 한 카테고리가 여러 축에 등장할 수도 있습니다.

### 즉각 효과 / 단번에 / Before→After
- **2. 뷰티** — "단번에 달라지는 피부" / before·after 클로즈업
- **7. 다이어트** — "눈에 보이는 변화" / 체형 비포애프터
- *흡수 포인트*: before/after 구도, "달라집니다" 결과형, 단번에·즉각 어휘

### 매력 / 외형 변화 / 욕구
- **1. 패션** — "입는 순간, 분위기가" / 즉각 매력
- **2. 뷰티** — "광채·결" 어휘 / 외형 업그레이드
- *흡수 포인트*: 즉각성, "순간·단번" 어휘, 모델 착용·사용 컷

### 신뢰 / 안심 / 까다로운 기준

## 콜라주 워크 예시

### 케이스: 단백질 쉐이크 D2C (다이어트 + 건기식 + 식품 콜라주)
랜딩 핵심: 한 끼 대용 + 단백질 30g + 1+1 첫 구매 혜택 + 4주 만족도 1위.

**픽업한 카드:**
- [효과/결과] 7번 다이어트에서 "눈에 보이는 변화" 결과형 + before/after 어휘
- [편의] 4번 식품에서 "이 한 끼면 충분합니다" 간편 어휘
- [신뢰/1위] 3번 건기식에서 "1위·임상" 권위 패턴
- [혜택] 모든 카테고리 공통 "1+1·첫 구매 X% OFF" 강조
- [CTA] 3번 "지금 시작하기"보다 랜딩 1차 행동인 **"지금 구매하기"** 채택 (사유 메모)
- [A/B 축] 7번 *소구 — A: 결과 / B: 과정* 채택, 또는 *이미지 — A: before/after / B: after 만족*

**가능한 헤드라인 변주:**
- "한 끼 대신, 단백질 30g" (4 간편 + 7 결과 + 정량)
- "4주 만에 핏이 바뀌었어요" (7 결과 + 후기형)
- "1+1, 오늘만 첫 구매 50%" (혜택 강조 + 시간 압박)
- "8천 명이 선택한 한 끼" (3 신뢰 정량 + 4 간편)

같은 입력에서도 카드 조합 비율을 바꾸면 매번 다른 변주가 나옵니다.

### 케이스: 신선 한우 D2C (식품 + 가성비 + 신뢰 콜라주)
랜딩 핵심: 산지직송 + 1+++ 등급 + 첫 구매 30% 할인 + 후기 1만 건.

**픽업한 카드:**
- [신선/원물] 4번 식품 "산지직송·신선" 어휘
- [신뢰] 3번 건기식 "1위·검증" 패턴을 *등급·후기 수치*로 응용
- [혜택] 첫 구매 % 강조
- [감정] 4번 "오늘 밥상이 달라집니다" 가족 식탁 어휘
- [CTA] 4번 "지금 주문하기"

**가능한 헤드라인 변주:**
- "1+++ 등급, 산지직송" (4 신선 + 등급 정량)
- "오늘 식탁에 1+++ 한우" (감정 + 정량)
- "후기 1만 명의 선택" (사회적 증거 수치형)

## 출력 형식 (카피라이터 → 디자이너 핸드오프)

카피라이터는 위 패턴을 흡수한 뒤, 디자이너에게는 **자연어 디자인 prompt** 형식으로 전달합니다. JSON 표가 아닙니다. 다음 요소를 자연스럽게 녹여 쓰세요:

```
[메인 카피] (헤드라인 9~22자, 카테고리 패턴 변형)
[서브 카피] (보조 설명 1~2줄, 선택)
[이미지 디렉션] (실사 — 커머스는 거의 실사. 인물/제품/공간, 성별/연령대/표정/포즈, 조명/톤/배경, 구도, dominant 컬러 톤(HEX), 실제 랜딩 제품 컷 합성 워크플로우 의무 명시)
[CTA 버튼] (카테고리에서 지정된 텍스트 우선 — 커머스는 대부분 "지금 구매하기"·"구매하기")
[톤·무드] (밝음/차분/긴급/감성/전문 중 골라 한 줄 묘사)
[A/B 변형] (카테고리에서 지정한 축으로 변형 2개. 변형 B에서 무엇이 어떻게 달라지는지 한 줄 설명)
```

이 prompt는 디자이너가 그대로 읽고 시각화할 수 있어야 합니다. 추상어 금지, 구체 명사·동작·표정 위주로.
