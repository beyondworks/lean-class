# YouTube Content Expander

YouTube 영상을 분석하여 스레드, 숏폼(60초), 롱폼(40분) 콘텐츠로 확장하고, 커뮤니티 유입 퍼널을 설계하는 스킬.

## 핵심 목표

1. **조회수/구독자 증가**: 다양한 포맷의 콘텐츠로 도달 범위 확대
2. **커뮤니티 구축**: 과제 기반 학습 커뮤니티로 유입
3. **1×N 인사이트**: 참여자들이 서로의 과제를 보며 배수의 학습 효과

## 커뮤니티 순환 구조

```
영상 시청 → 과제 수행 → 크리에이터 검수 → 커뮤니티 공유 → 1×N 인사이트 획득 → 재참여
```

**핵심 원칙:**
- 참여자에게 실질적 이득 제공 (업무 활용, 수익화)
- 숏폼/롱폼은 커뮤니티 유입 퍼널
- 크리에이터의 화법/말투/행동 일관성 유지

## 워크플로우

### Step 1: YouTube 영상 분석

YouTube URL이 제공되면:

1. **영상 정보 추출**: 제목, 설명, 채널명, 조회수, 댓글 반응 (웹 검색 활용)
2. **트랜스크립트 추출**: 자막/스크립트 확보
3. **핵심 메시지 파악**: 주제, 핵심 인사이트, 타겟 오디언스

```
분석 체크리스트:
□ 영상의 핵심 주제는?
□ 타겟 시청자는 누구인가?
□ 어떤 문제를 해결하는가?
□ 시청자가 얻는 가치는?
□ 댓글에서 반응이 좋은 부분은?
```

## 인기 영상 리서치 기반 썸네일/제목 기획

사용자가 유튜브 리뷰 영상의 썸네일 문구/영상 제목을 요청하면서 “국내/해외 인기 콘텐츠를 분석해줘”, “조회수 폭발”, “후킹 텍스트”를 요구하면, 단순 브레인스토밍 전에 실제 유튜브 제목 데이터를 수집한다.

권장 절차:
1. `yt-dlp --dump-json --skip-download --flat-playlist 'ytsearch25:<query>'`로 국내/해외 관련 검색어를 여러 개 돌려 최소 50~100개 제목을 수집한다.
2. `view_count`, 채널, 제목, 검색어를 중복 제거 후 조회수순으로 정렬한다.
3. 제목 패턴을 분류한다: 브랜드명 전면, 숫자/리스트형, 직접 써봄/리뷰형, 시간·돈·업무 성과형, 충격/끝판왕형, 진입장벽 제거형, 비교/대립형.
4. 제공된 썸네일 이미지가 있으면 시각 요소를 먼저 읽고, 텍스트는 이미지 구도에 맞춰 5~9자 중심으로 제안한다. 긴 설명은 영상 제목으로 보내고 썸네일에는 짧은 감정/대립/성과 단어를 둔다.
5. 최종안은 `썸네일 메인 문구 + 보조 문구 + 영상 제목` 조합으로 3개 이상 제시하고, 추천 1순위를 명확히 고른다.

예시 패턴:
- `AI 비서 끝판왕` + `Genspark Claw 써봤습니다. OpenClaw가 웹으로 들어오면 생기는 일`
- `OpenClaw 웹버전?` + `설치 없이 쓰는 OpenClaw? Genspark Claw 실사용 리뷰`
- `이건 챗봇이 아님` + `Genspark Claw 공개: 챗봇이 아니라 ‘AI 직원’에 가까운 이유`


## 썸네일·제목 기획 추가 워크플로우

AI 도구 리뷰, AI 에이전트 리뷰, 신제품 론칭/비교 영상의 썸네일 문구와 영상 제목을 기획할 때는를 먼저 참고한다.

핵심 절차:
1. 국내/해외 유사 주제 인기 영상 50–100개 이상을 수집한다.
2. 조회수 상위 제목을 브랜드명, 숫자/리스트, 에이전트/자동화, 성과/시간절약, 강한 감탄, 직접 써봄, 진입장벽 제거 패턴으로 분류한다.
3. 첨부 썸네일이 있으면 이미지 구도, 로고 위치, 얼굴 표정, 텍스트 여백을 먼저 분석한다.
4. 썸네일은 감정적 클릭 이유를 4–8단어 안에 담고, 영상 제목은 구체적 제품명·검증 각도·사용 결과를 보완한다.
5. 최종안은 너무 많은 후보만 나열하지 말고 1순위 조합을 명확히 추천한다.

## 품질 체크리스트

- [ ] 크리에이터의 화법/말투와 일관성 있는가?
- [ ] 숏폼이 롱폼/커뮤니티로의 유입 동기를 부여하는가?
- [ ] 과제가 참여자에게 실질적 가치를 제공하는가?
- [ ] 1×N 인사이트 구조가 명확한가?
- [ ] CTA가 자연스럽게 녹아있는가?
- [ ] AI 도구 리뷰 썸네일/제목이라면 인기 콘텐츠 표본 기반의 후킹 패턴을 반영했는가?
- [ ] 썸네일 문구와 영상 제목의 역할을 분리했는가? 썸네일은 감정, 제목은 구체성과 검색성을 맡는다.
