# n8n Workflow Patterns

검증된 n8n 워크플로우 아키텍처 패턴 가이드.

## 5가지 핵심 패턴
1. **Webhook Processing** (35%) - HTTP 요청 수신 → 처리 → 출력
2. **HTTP API Integration** (45%) - REST API 호출 → 변환 → 활용
3. **Database Operations** (28%) - DB 읽기/쓰기/동기화
4. **AI Agent Workflow** - AI 에이전트 + 도구 + 메모리
5. **Scheduled Tasks** (28%) - 반복 자동화 워크플로우
