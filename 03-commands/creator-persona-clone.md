# Creator Persona Clone

YouTube 영상 URL을 분석하여 크리에이터의 화법/어조/스타일을 심층 분석하고,
재사용 가능한 페르소나 클론을 생성합니다.

## 모드
1. **분석 모드**: YouTube URL 제공 → 크리에이터 분석 리포트 + 페르소나 프로필 + 시스템 프롬프트 생성 → Notion 저장
2. **대본 모드** (v1.1 예정): 기존 프로필 기반 → 롱폼/숏폼 대본 작성

## 입력
- YouTube 영상 URL 1~5개 (같은 크리에이터, 최소 3개 권장)

$ARGUMENTS
